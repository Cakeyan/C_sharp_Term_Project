# Client所有函数参数接口

- 点击文件后再预览不用拖动
- `namespace`都是Client
- 参数空着的是变量
- 构造函数一般和私有变量、函数一般不列出
- 类名和文件名是一样的，所有文件都只有一个类
- 类型为 **鼠标点击事件** 的，参数一般是`object sender, RoutedEventArgs e`，一般是接收一个鼠标点击
